# KotlinDataStructuresLab

Este proyecto contiene la solución a 30 desafíos de programación en Kotlin enfocados en el manejo de estructuras de datos (Arrays, Lists, Sets, Maps, Matrices).

**Estudiante:** Mateo Betancur Escobar
**Curso:** Desarrollo de Aplicaciones Móviles con Android

## Lista de Retos Solucionados

1.  **Control de Gastos Diarios:** Cálculo manual de total, promedio, máximo y mínimo.
2.  **Filtro de Empleados Senior:** Filtrado por edad y transformación a mayúsculas.
3.  **Limpieza de Contactos Duplicados:** Uso de `Set` para unicidad y ordenamiento.
4.  **Análisis de Carrito de Compras:** Frecuencia de productos con `Map`.
5.  **Historial de Navegación Reversible:** Inversión manual de un Array.
6.  **Compatibilidad de Intereses (Match):** Intersección y diferencia de conjuntos.
7.  **Inventario de Despensa:** Gestión dinámica de inventario con `MutableMap`.
8.  **Cálculo de Promedio Académico:** Promedio ponderado de notas.
9.  **Traductor de Códigos de País:** Iteración sobre estructuras de clave-valor.
10. **Rotación de Turnos de Trabajo:** Desplazamiento circular de elementos.
11. **Organizador de Maleta:** Agrupamiento por atributos (peso).
12. **Verificador de Ruta Reversible:** Lógica de palíndromos sobre coordenadas.
13. **Reparto de Pedidos por Camión:** División de listas en lotes (`chunked`).
14. **Identificador de Ticket Único:** Búsqueda del primer elemento no repetido.
15. **Calculadora de IVA Selectiva:** Aplicación condicional de impuestos.
